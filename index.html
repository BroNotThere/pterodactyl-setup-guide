🚀 Install Pterodactyl

Run the single command below on a fresh Ubuntu 20.04 / 22.04 or Debian server.

bash <(curl -s https://pterodactyl-installer.se)
Tip ▸ The installer will ask which panel + wings version you want; choose “Latest” for both unless you need a specific release.

🌐 Expose SSH (Playit Tunnel)

Paste the entire block in your terminal. It will add the Playit PPA and start the service.

curl -SsL https://playit-cloud.github.io/ppa/key.gpg | gpg --dearmor | sudo tee /etc/apt/trusted.gpg.d/playit.gpg >/dev/null
echo "deb [signed-by=/etc/apt/trusted.gpg.d/playit.gpg] https://playit-cloud.github.io/ppa/data ./" | sudo tee /etc/apt/sources.list.d/playit-cloud.list
sudo apt update
sudo apt install playit
sudo systemctl start playit
sudo systemctl enable playit
playit setup
Next step ▸ The playit setup command prints a claim‑link. Open it in your browser, create a TCP + UDP tunnel, set Port 22, and copy the IP + Port combo. That becomes your public SSH endpoint.
